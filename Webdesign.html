Bezochte sites bij het bouwen van Cloud Field

W3Schools

Allround webdesign/programmeer hulp site. Onder andere gebruikt bij het aanroepen van file locaties, het vinden van mooie CSS templates en het toevoegen van een sidebar menu.

Developer.mozilla.org

Tips van de browser Mozilla Firefox voor beginnende webdesigners. Beschrijft de benodigde eerste stappen voor een eigengemaakte website en de mogelijkheden voor het publiceren van een website.

Github pages

Gebruiksaanwijzing voor Github pages, een manier om gratis je website te publiceren. Beschrijft hoe je een github project aan te maken en deze te publiceren via github pages.

Stackoverflow

Vraagbaak voor programmeurs. In dit geval gebruikt om een manier te vinden voor het laten zien van txt bestanden met gedichten op de Gedichten pagina.

Visual Studio Code

Een open source code editor van Microsoft. Basis code editor met kleurmarkering, compatible met vele programmeertalen.

Mediacollege

Onderwijsplatform voor elektronische media. In dit geval gebruikt om te leren hoe een 'random' afbeelding uit een reeks afbeeldingen te laten zien op de Wolken pagina.

Jekyllrb

Platform om websites en blogs te faciliteren. Ik gebruik het om lokaal de website te deployen (jekyll serve) en te testen. Of je moet de voorkeur hebben om dingen direct live te zetten.
